# 🎮 Прокачай Речь

Обучающая web-игра для развития дикции у детей 6-11 лет.

## 🚀 Быстрый старт

```bash
# Запуск проекта (нужен Python)
cd PC_aDiction/src
python -m http.server 8000

# Открыть в браузере
http://localhost:8000
```

## 📁 Структура проекта

```
PC_aDiction/
├── src/
│   ├── index.html          # Главная страница
│   ├── index.js            # Основная логика
│   ├── styles/
│   │   └── main.css        # Стили
│   ├── components/         # Компоненты игры
│   ├── assets/
│   │   ├── audio/          # Аудио файлы
│   │   └── images/         # Изображения
│   └── utils/              # Утилиты
├── server/                 # Бэкенд (Whisper)
├── public/                 # Статические файлы
└── CLAUDE.md              # Документация для Claude
```
